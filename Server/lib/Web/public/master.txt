CherryKKuTu 운영 정책

제 1조 (제재 사항)

1. 제재 사항

{ 
   1. 욕설 사용 (경고 2회 지급),
   2. 패드립, 섹드립(채팅 정지 ( 구현 예정 )),
   3. 서버를 비하하는 행위 ( ex : 체리끄투 노잼 ) (경고 1회 지급),
   4. 타 프리서버(jjo.kr/kkutu에 기재 되지 않은 끄투 서버는 제외)를 지속적으로 언급 또는 홍보하는 행위
   5. 유처처리부가 상황에 따라 제재가능 (경고 1~3회 지급),
   6. 분쟁 유도 (1차 7일밴, 2차 30일밴, 3차 영구 IP 밴),
   7. 유저끼리 소위감 느끼는 행위 (1차 경고2회, 2차 30일 밴, 3차 ip 밴),
   지속적으로 추가 예정.
}, 게스트{
   1. 욕설 사용 (ip밴),
   2. 패드립 , 섹드립 (1항 동일)
   3~7. 1항 동일.
}





정보 수집 안내

체리끄투는 채팅, 아이피 등을 *.log 파일로 끄투 접속시 자동으로 수집 됩니다.
이 파일은 욕설, 디도스 등 체리끄투에 운영정책에 위반 되는 행위, 안전성을 위해 조회, 사용됩니다.
이 파일은 마지막 수정일로 부터 2년이 지나면 자동으로 삭제(또는 파기)됩니다.
