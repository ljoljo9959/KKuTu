CherryKKuTu 운영 정책

제 1조 (제재 사항)

1. 제재 사항

{ 
   1. 욕설 사용 (경고 2회 지급),
   2. 패드립, 섹드립(채팅 정지),
   3. 서버를 비하하는 행위 ( ex : 체리끄투 노잼 ) (경고 1회 지급),
   4. 타 프리서버(모든 끄투 오픈소스를 사용 한 서버)를 언급 또는 홍보하는 행위 (경고 2~3회 지급)
   5. 유처관리부가 상황에 따라 제재가능 (경고 1~3회 지급),
   6. 분쟁 유도 (1차 7일밴, 2차 30일밴, 3차 영구 IP 밴),
   7. 유저끼리 소위감 느끼는 행위 (1차 경고2회, 2차 30일 밴, 3차 ip 밴),
   8. 허위 신고 , 허위 단어 요청(경고 3회)
   지속적으로 추가 예정.
};

제 2조 단어 규정

{
   BFKKUTU 단어 규정을 일부 사용 하였습니다.
   한방 단어 : 방어 단어와 1/5 비율로 맞춰 가며 추가한다.
   (방어 단어가 5개라면 한방단어 1개 추가 할 수 있다.)
   기호 표시 : 기호는 빼고 표시한다. 단 한국어로 기호를 읽혀 제목이 되어 있다면 그대로 추가한다.
   글자 수 제한 : 최대 59자.

   영어 {
      등재는 소문자로만 등재한다. (띄어쓰기 , 기호 X)
      (tasdofidk dkfdjpqk$ => tasdofidkdkfdjpqk)
      네이버 영어사전, 워드넷, 윅셔너리만 등재한다. (뜻이 없다면 등재하지 않는다.)
      번역기 뜻과 웹수집은 등재 하지 않는다.
   }

   노인정 : 표준대국어사전 , 네이버 지식백과, 우리말샘
   문학 책명 : 네이버책(사진이 존재 해야만 함)(따라서 yes24 가능)
   만화책, 라노벨 : 알라딘(따라서 yes24 가능)
   만화책 : 웹사이트(카카오페이지 등)
   애니메이션 : 국내 정발 애니, 애니 극장판
   만화 / 애니메이션 / 웹툰 , 라노벨 , 게임 : 국내 정발 이름을 사용 한다.
   웹툰 : 네이버, 다음, 네이트, 레진 코믹스, 배틀 코믹스, GT코믹 등 공식 웹툰 사이트에서 연재중 혹은 연재완료한 웹툰
   역사 : 표준국어대사전, 네이버 지식백과, 시호
   음식 : 실제 음식 이름
   철도역 : 위키백과, 나무위키 (개통된 역만 등재한다.)
   게임명 : 네이버 게임정보, 위키백과, 일반 놀이, 플래시 게임, 스팀게임 (서비스 종료된 게임도 가능하다, 플래시 게임은 웹사이트만 등재한다(예 : 충치 디펜스))
   신조어 : 위키백과, 나무위키
   엘소드 : 캐릭터, 전직, 스킬, NPC, 마을, 몬스터, 칭호, 위키, 인게임
   디지몬 : 방영 목록, 디지몬, 등장인물
   포켓몬스터 : 포켓몬, 기술, 특성, 지명, 인물
   리그오브레전드 : 챔피언, 스킨, 스킬, 아이템
   오버워치 : 영웅, 스킨, 맵, 인벤, 인게임
   히어로즈 오브 더스톰 : 영웅, 스킬, 특성, 스킨, 전장, 인벤, 인게임
   러브라이브 : 캐릭터, 카드, 인벤
   도타2 : 인게임 (아이템, 화면 등등), 위키
   영화 : 국내 개봉작, 네이버/다음 영화
   유명인 : 네이버, 구글 (위키 안됨.) => 프로필이 나와야 함.
   하이큐 : 시리즈 내 등장인물, 시리즈 내 출현하는 모든 학교, 방영목록, 극장판, 위키

}

제 3조 제휴 기준{

   1.  제휴 서버는 jjo.kr/kkutu 또는 kkutu.kr 에 등록 되어야 한다.
   2. 제휴 서버는 끄투 레포지토리를 따라서 G P L 3 . 0을 따라야 한다 ( 이하 위반 해서는 안된다 )
   3. 제휴 서버는 최종적으로 체리끄투 디스코드에서 2일간 공개 투표후 결정 되어야 한다.
   4. 제휴 서버는 서버에 부적절한 것이 발견 되지 않아야 한다. ( 예시 : 19금 광고 표기 )

}
